## 设计原则

### 核心原则

1. **完全公开访问**
   - 所有已发布的专题对所有人开放（包括未登录访客）
   - 访客可自由浏览专题内容
   - AI 使用与管理操作需要登录

2. **统一网站模型**
   - 所有专题都是网站形式，通过 AI Agent 辅助创建
   - 无需区分知识库型和网站型

3. **简化设计**
   - 移除任务/提交/评价工作流
   - 专注核心的"创建-浏览-AI辅助"流程

4. **用户友好**
   - 访客友好：无需注册即可浏览
   - 简单直接：无复杂流程
   - 按需认证：只在必要时要求登录
