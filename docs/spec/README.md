## 设计原则

### 核心原则

1. **完全公开访问**
   - 所有已发布的专题对所有人开放（包括未登录访客）
   - 访客可自由浏览专题和下载资源
   - 只有提交成果需要登录

2. **简化设计**
   - 无加入机制
   - 无复杂的任务系统
   - 专注核心的"创建-浏览-提交-评价"流程

3. **用户友好**
   - 访客友好：无需注册即可浏览
   - 简单直接：无复杂流程
   - 按需认证：只在必要时要求登录
