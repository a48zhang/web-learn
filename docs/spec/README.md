## 设计原则

1. **完全公开访问** — 已发布专题对所有人开放（包括未登录访客）
2. **统一网站模型** — 所有专题都是网站形式，通过 AI Agent 辅助创建
3. **简化设计** — 移除任务/提交/评价工作流，专注核心流程
4. **用户友好** — 访客友好、简单直接、按需认证
