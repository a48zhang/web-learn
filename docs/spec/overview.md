# 产品概述

> 最后更新：2026-04-03

## 产品定位

**专题学习平台** - 让教师能够快速搭建和管理专题学习网站系统。

"专题学习"是教育领域的专业概念，强调：
- **主题性**：围绕特定学习主题展开深度探究
- **整合性**：跨学科知识融合，综合性学习
- **探究性**：学生主动参与、协作学习
- **成果导向**：最终产出学习成果并获得评价

## 执行摘要

本产品是一个面向教育领域的专题学习平台，支持教师快速创建专题学习空间（知识库型或网站型），学生和访客可以自由浏览专题内容。系统集成学习助手 AI 帮助理解专题内容，专题搭建 AI 帮助教师创建和编辑专题。

## 目标用户

**主要用户：**
- **教师**：创建和管理专题学习空间，使用 AI 助手搭建专题内容
- **学生**：浏览专题内容，使用 AI 助手辅助学习理解
- **访客**：浏览公开的专题内容（未登录用户）

## 适用场景

- 课堂内外的混合式教学
- 项目式学习（PBL）
- 跨学科综合学习
- 研究性学习课程
- 校本课程实施

## 核心价值

1. **快速部署**：教师可快速创建专题学习网站，无需技术背景
2. **双模式支持**：知识库版（模板化）和网站版（自定义代码）
3. **AI 辅助搭建**：专题搭建 Agent 帮助教师生成内容、优化结构
4. **AI 学习助手**：学习助手 Agent 帮助学生理解专题内容、答疑解惑
5. **完全公开**：所有已发布专题对所有人开放，降低参与门槛
6. **访客友好**：无需注册即可浏览专题内容

## 专题空间类型

### 知识库版

**特点：**
- 使用预定义的固定模板
- 教师只需上传每页的内容（文本、图片、链接等）
- 内容存储在数据库中
- 适合结构化的学习内容

**优势：**
- 无需技术背景，操作简单
- 统一的展示样式
- 便于管理和维护
- 支持内容搜索和索引

**适用场景：**
- 课程讲义
- 学习指南
- 知识体系梳理
- 结构化教学内容

### 网站版

**特点：**
- 教师上传完整的网站代码（HTML/CSS/JS）
- 代码存储在对象存储（OSS）上
- 高度自定义的展示方式
- 适合需要复杂交互的专题

**优势：**
- 完全自定义的外观和功能
- 支持复杂的交互逻辑
- 可集成第三方工具
- 更灵活的展示方式

**适用场景：**
- 交互式学习应用
- 实验演示网站
- 项目作品展示
- 需要特殊功能的专题

## 公开访问设计

**核心原则：**
- ✅ 所有已发布的专题对**所有人完全公开**（包括未登录访客）
- ✅ 访客无需登录即可浏览专题内容
- ✅ AI 学习助手需登录后使用
- ✅ 专题管理需登录（教师身份）

**访问分级：**
- **浏览专题**：所有人可访问（包括访客）
- **AI 学习助手**：需登录（学生和教师）
- **专题管理**：需登录（教师身份）
- **AI 专题搭建**：需登录（教师身份）
