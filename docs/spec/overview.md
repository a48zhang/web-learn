# 产品概述

> 最后更新：2026-04-10

## 产品定位

**专题学习平台** - 让教师通过 AI 辅助快速搭建专题学习网站。

"专题学习"是教育领域的专业概念，强调：
- **主题性**：围绕特定学习主题展开深度探究
- **整合性**：跨学科知识融合，综合性学习
- **探究性**：学生主动参与、协作学习
- **内容驱动**：围绕主题组织高质量的学习内容，供自主探究

## 执行摘要

本产品是一个面向教育领域的专题学习平台，教师通过 AI Agent 辅助对话式地创建和编辑专题网站。系统集成学习助手 AI 帮助理解专题内容，专题搭建 Agent 帮助教师通过对话生成网站。

**系统架构：** 微服务架构（Gateway + Auth + Topic Space + AI），通过 API Gateway 统一入口，服务独立部署和扩缩容。

## 目标用户

**主要用户：**
- **教师**：通过 AI Agent 对话创建和编辑专题网站
- **学生**：浏览专题内容，使用学习助手辅助理解
- **访客**：浏览公开的专题内容（未登录用户）

## 适用场景

- 课堂内外的混合式教学
- 项目式学习（PBL）
- 跨学科综合学习
- 研究性学习课程
- 校本课程实施

## 核心价值

1. **AI 驱动创建**：教师通过对话描述需求，Agent 生成网站代码，无需技术背景
2. **即时预览**：WebContainer 在浏览器中运行，代码修改实时可见
3. **AI 学习助手**：帮助学生理解专题内容、答疑解惑
4. **完全公开**：所有已发布专题对所有人开放，降低参与门槛
5. **访客友好**：无需注册即可浏览专题内容
6. **微服务架构**：服务独立部署和扩缩容，技术栈现代化

## 专题网站编辑器

所有专题统一为网站形式，通过编辑器界面创建和编辑：

**特点：**
- 三栏可调整布局（文件树 + Agent对话 + 浏览器预览）
- Agent 主动协作：先询问偏好（风格、布局、颜色等），再生成代码
- 支持多种输入方式：对话输入、文件上传（Markdown/Word/PDF）、混合方式
- WebContainer 在后台运行，iframe 嵌入实时预览
- 支持常见前端框架（React、Vue、Svelte 等）
- 双击文件可打开 Monaco 编辑器直接修改

## 公开访问设计

**核心原则：**
- ✅ 所有已发布的专题对**所有人完全公开**（包括未登录访客）
- ✅ 访客无需登录即可浏览专题内容
- ✅ AI 学习助手需登录后使用
- ✅ 专题管理需登录（教师身份）

**访问分级：**
- **浏览专题**：所有人可访问（包括访客）
- **AI 学习助手**：需登录（学生和教师）
- **专题管理**：需登录（教师身份）
- **AI 专题搭建**：需登录（教师身份）
