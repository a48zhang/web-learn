# 用户角色与权限

> 最后更新：2026-04-03

## 角色定义

### 访客（未登录用户）

**描述：** 访问平台的任何人，无需注册登录

**主要功能：**
- 查看所有已发布的专题学习空间
- 查看专题详情和页面内容
- 注册账号

### 学生

**描述：** 已注册的专题学习参与者

**主要功能：**
- 所有访客权限
- 使用 AI 学习助手帮助理解专题内容
- 需要登录认证

### 教师

**描述：** 专题学习的组织者和指导者

**主要功能：**
- 所有访客权限
- 创建和管理专题学习空间（知识库型或网站型）
- 使用 AI 学习助手回顾专题内容
- 使用 AI 专题搭建助手创建和编辑专题
- 需要登录认证

## 权限矩阵

### 专题管理权限

| 操作 | 访客 | 学生 | 教师 |
|------|:----:|:----:|:----:|
| 查看已发布专题列表 | ✓ | ✓ | ✓ |
| 查看专题详情 | ✓ | ✓ | ✓ |
| 创建专题 | ✗ | ✗ | ✓ |
| 编辑专题 | ✗ | ✗ | ✓ 自己创建的 |
| 发布/关闭专题 | ✗ | ✗ | ✓ 自己创建的 |

### AI Agent 权限

| Agent 类型 | 访客 | 学生 | 教师 |
|------|:----:|:----:|:----:|
| 学习助手 Agent（learning） | ✗ | ✓ 所有已发布的 | ✓ 所有已发布的 |
| 专题搭建 Agent（building） | ✗ | ✗ | ✓ 自己创建的专题 |

## 权限说明

### 访客权限（完全公开）

- 可以查看所有已发布的专题列表
- 可以查看任何已发布专题的详情和页面内容
- **无需登录即可浏览专题内容**

### 学生权限

- 所有访客权限
- 可以使用 AI 学习助手理解专题内容、查询进度
- 需要登录认证

### 教师权限

- 所有访客权限
- 可以创建多个专题学习空间（知识库型或网站型）
- 对自己创建的专题拥有完全管理权
- 可以使用 AI 学习助手回顾专题内容
- 可以使用 AI 专题搭建助手创建和编辑专题
- 需要登录认证

## 公开访问原则

### 核心原则

1. **完全公开浏览**
   - 所有已发布的专题对所有人公开可见
   - 访客无需登录即可自由浏览
   - 降低参与门槛，促进知识共享

2. **分级操作权限**
   - 浏览专题：公开访问（访客、学生、教师）
   - AI 学习助手：需要登录（学生、教师）
   - 专题管理：需要教师身份
   - AI 专题搭建：需要教师身份

3. **无加入机制**
   - 不需要"加入"或"订阅"操作
   - 所有已发布专题可直接浏览
   - 简化参与流程

### 设计理念

- **开放优先**：默认公开，促进知识传播
- **简单直接**：访客即可浏览，无需复杂流程
- **AI 辅助**：学习助手帮助理解，搭建助手帮助创建
- **按需认证**：只在需要 AI 或管理操作时要求登录
